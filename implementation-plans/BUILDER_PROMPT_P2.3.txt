## Tutorial Structure

### Section 1: Install (30 seconds)

```bash
# Clone and build
git clone https://github.com/davidnichols-ops/aafp.git
cd aafp/implementations/rust
cargo build --release

# The CLI is now at ./target/release/aafp
# Add to PATH (optional):
export PATH="$PWD/target/release:$PATH"
```

Expected output: `Finished release [optimized] target(s)`

### Section 2: Your First Agent (1 minute)

```bash
# Start an echo agent (responds with whatever you send it)
aafp serve --capability echo
```

Expected output:
```
Agent ID:     a3f4...
Serving on:   127.0.0.1:12345
Capabilities: echo
Press Ctrl+C to stop
```

### Section 3: Call an Agent (30 seconds)

Open a new terminal:

```bash
# Call the echo agent
aafp call echo "hello"
```

Expected output:
```
Response: hello
```

### Section 4: Build Your Own Agent (2 minutes)

Show a 3-line Rust agent:

```rust
// Cargo.toml: aafp-sdk = { path = "implementations/rust/crates/aafp-sdk" }

use aafp_sdk::simple::{Agent, Request, Response};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    Agent::serve()
        .capability("greet")
        .handler(|req: Request| async move {
            Ok(Response::text(format!("Hello, {}!", req.body())))
        })
        .start()
        .await?;
    Ok(())
}
```

Then call it:
```bash
aafp call greet "World"
# Response: Hello, World!
```

### Section 5: Python Agent (1 minute)

```python
# pip install aafp  (when available — for now, use the Rust API)

from aafp import Agent

agent = Agent.serve(capability="greet")

@agent.handler
async def greet(request):
    return {"text": f"Hello, {request['text']}!"}

await agent.start()
```

### Section 6: Docker (30 seconds)

```bash
docker compose up
# Starts 3 agents that can discover and talk to each other
```
