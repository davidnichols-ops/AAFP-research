## New Commands to Add

### 1. `aafp serve` — Start serving in one command

```bash
aafp serve --capability echo
aafp serve --capability translation --bind 0.0.0.0:4433
aafp serve --capability ocr --identity my-key.bin
```

This wraps the P2.1 simple API:
```rust
use aafp_sdk::simple::{Agent, Request, Response};

Agent::serve()
    .capability("echo")
    .handler(|req: Request| async move {
        Ok(Response::text(req.body().to_string()))  // echo
    })
    .start()
    .await?;
```

**Implementation:**
- Create `crates/aafp-cli/src/commands/serve.rs`
- Use `Agent::serve()` from `aafp_sdk::simple`
- `--capability` can be repeated (multiple capabilities)
- `--bind` optional (default: `0.0.0.0:0`)
- `--identity` optional (path to keypair file, auto-generates if not provided)
- Default handler: echo (returns the request body as response)
- Print agent ID, bind address, and capabilities on startup
- Keep running until Ctrl+C
- Print connection logs as they happen

### 2. `aafp call` — Discover and call in one command

```bash
aafp call echo "hello"
aafp call translation "translate this" --identity my-key.bin
aafp call echo "hello" --json
```

This wraps the P2.1 simple API:
```rust
use aafp_sdk::simple::{Agent, Request};

let mut agent = Agent::connect().connect().await?;
let result = agent.discover("echo")
    .call(Request::text("hello"))
    .await?;
println!("{}", result.body());
```

**Implementation:**
- Create `crates/aafp-cli/src/commands/call.rs`
- Use `Agent::connect()` from `aafp_sdk::simple`
- First arg: capability name
- Second arg: message text
- `--identity` optional
- `--json` flag: output as JSON `{"capability": "echo", "request": "hello", "response": "echo: hello"}`
- Print the response body to stdout
- If no agents found, print helpful error: "No agents with capability 'echo' found. Is an agent serving on this network?"

## P2.1 API Reference (from simple.rs)

```rust
// Serving
aafp_sdk::simple::Agent::serve()
    .capability("echo")           // can be called multiple times
    .handler(|req| async move { Ok(Response::text(req.body())) })
    .bind(SocketAddr)             // optional, default 0.0.0.0:0
    .with_keypair(AgentKeypair)   // optional, auto-generates
    .start().await                // → Result<ServingAgent, SdkError>

// ServingAgent
serving_agent.id()        // → &AgentId
serving_agent.addr()      // → &str (bind address)
serving_agent.record()    // → &AgentRecord
serving_agent.stop()

// Connecting
aafp_sdk::simple::Agent::connect()
    .with_keypair(AgentKeypair)   // optional
    .with_seeds(Vec<String>)      // optional
    .connect().await              // → Result<ConnectedAgent, SdkError>

// ConnectedAgent
connected_agent.discover("echo")  // → DiscoveryBuilder
connected_agent.register(&record) // populate DHT
connected_agent.id()              // → &AgentId

// DiscoveryBuilder
discovery.call(Request::text("hello")).await  // → Result<Response, SdkError>

// Request
Request::text("hello")    // → Request
Request::data(vec![...])  // → Request
req.body()                // → &str
req.payload()             // → Option<&[u8]>

// Response
Response::text("world")   // → Response
Response::data(vec![...]) // → Response
resp.body()               // → &str
resp.payload()            // → Option<&[u8]>
```
